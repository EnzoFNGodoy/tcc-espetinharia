-------------------------------------------------------------------------------------------------------

  O projeto consiste em um sistema de gerenciamento de bares e restaurantes com o intuito de automatizar 
as operações de rotina, como por exemplo realizar cadastro de clientes, de vendas e até mesmo de entradas
e saídas de estoque. Com essas informações registradas no banco o usuário com as permissões necessárias
poderá consultá-las, alterá-las e excluí-las.
